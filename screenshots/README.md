# Screenshots Directory

Please place your application screenshots in this directory with the following filenames:

1. `login.png` — Login screen
2. `dashboard.png` — Sales dashboard screen with statistics cards
3. `customers.png` — Customer list screen showing pagination and search
4. `reports_bar.png` — Reports screen showcasing the monthly revenue bar chart
5. `reports_line.png` — Reports screen showcasing the monthly orders line chart

Once placed, these will automatically be displayed in the main `README.md` file.
